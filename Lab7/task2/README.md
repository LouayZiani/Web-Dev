# Lab 7 Task 2: OOP in Python

## This project demonstrates basic Object-Oriented Programming concepts in Python.

## Implemented Concepts
- Base class: `Animal`
- Child classes: `Dog`, `Cat`
- Inheritance
- Method overriding
- Polymorphism
- Modules and imports

## File Structure
- `models.py` - contains class definitions
- `main.py` - creates objects and demonstrates usage

## How to Run
```bash
python main.py
